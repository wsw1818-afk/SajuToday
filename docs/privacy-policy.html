개인정보처리방침

사주투데이 (SajuToday)

최종 업데이트: 2025년 1월

1. 개인정보의 처리 목적

사주투데이(이하 "앱")는 다음의 목적을 위하여 개인정보를 처리합니다.

  • 맞춤형 운세 서비스 제공
  • 사용자 생년월일시 기반 사주 계산
  • 앱 서비스 품질 개선

2. 수집하는 개인정보 항목

필수 정보

  • 이름: 사용자 식별 및 맞춤 서비스 제공
  • 성별: 성별에 따른 운세 해석
  • 생년월일: 사주 계산의 기본 정보
  • 양력/음력 구분: 정확한 날짜 변환

선택 정보

  • 출생 시간: 시주(時柱) 계산 (입력하지 않아도 서비스 이용 가능)
💡 중요: 수집된 모든 정보는 사용자 기기 내부(로컬 스토리지)에만 저장되며, 외부 서버로 전송되지 않습니다.

3. 개인정보의 보유 및 이용 기간

  • 개인정보는 앱이 기기에 설치되어 있는 동안 보유됩니다.
  • 앱 삭제 시 모든 데이터가 자동으로 삭제됩니다.
  • 사용자는 언제든지 앱 내 설정에서 데이터를 초기화할 수 있습니다.

4. 개인정보의 제3자 제공

앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

  • 모든 데이터는 기기 내부에서만 처리됩니다.
  • 외부 서버 전송, 분석, 광고 목적 사용이 없습니다.

5. 개인정보의 안전성 확보 조치

앱은 개인정보 보호를 위해 다음의 조치를 취하고 있습니다.

  • 기기 내 안전한 저장소(AsyncStorage/SecureStore) 사용
  • 네트워크 전송 없음 (오프라인 사용 가능)
  • 앱 삭제 시 모든 데이터 완전 삭제

6. 정보주체의 권리·의무

사용자는 다음과 같은 권리를 행사할 수 있습니다.

  • 개인정보 열람 요구
  • 오류 등이 있을 경우 정정 요구
  • 삭제 요구 (앱 설정 → 데이터 초기화)
  • 처리 정지 요구

7. 외부 API 사용

앱은 다음의 외부 API를 사용할 수 있습니다.

  • 한국천문연구원(KASI) API: 음력/양력 변환 목적으로만 사용. 생년월일만 전송되며, 개인 식별 정보는 전송되지 않음.

8. 아동의 개인정보 보호

앱은 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 만 14세 미만 아동이 부모 또는 법정대리인의 동의 없이 개인정보를 제공한 경우, 해당 정보는 즉시 삭제됩니다.

9. 개인정보처리방침 변경

본 개인정보처리방침은 법률 또는 서비스의 변경에 따라 변경될 수 있습니다. 변경 시 앱 내 공지 또는 업데이트 노트를 통해 안내합니다.
