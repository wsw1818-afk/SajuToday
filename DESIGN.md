# SajuToday Design System v1.0

> 한국 무속 점집 부적 분위기 — "오늘의 운세"를 받아보는 느낌
> 결정일: 2026-04-18 (Council 16번째 합의 + 사용자 직접 선택)

## 컨셉

**부적 (BUJEOK)** — 한국 전통 무속인이 써준 부적 종이를 받아보는 분위기.

- 베이지 한지 종이 위에 빨간 부적 인장 + 검은 먹글씨
- 큰 한자 "吉" 손글씨 부적 + 한국어 본문 풀이
- 카테고리는 부적 미니 도장 형태
- SNS 카드 그리드 ❌ / 점집 점괘 ✅

## 컬러 시스템

### Primary Palette (3색 원칙)

```
--color-paper      #F5E6C8   # 한지 베이지 (배경)
--color-bujeok     #C0392B   # 부적 적색 (강조/인장)
--color-ink        #1A1A1A   # 먹색 (본문 텍스트)
```

### Secondary Palette (필요시만)

```
--color-paper-dark  #E8D4A0   # 한지 어두운 영역 (구분선)
--color-paper-light #FFFEF5   # 카드 배경 (밝은 한지)
--color-ink-soft    #4A3B30   # 먹색 부드러운 (서브 텍스트)
--color-bujeok-soft #E8B4A8   # 부적 적색 흐림 (배경 강조)
--color-gold        #B8860B   # 황금 (특별 액센트, 길일 표시 등)
```

### 점수 색상 (4단계)

```
대길 (80+): #C0392B (부적 적색, 진함)
길  (60+): #D4732C (주황빛 적색)
평  (40+): #8B7355 (먹색 흐림)
흉  (40-): #4A3B30 (어두운 먹색)
```

### 명시적 폐기

❌ ThemeContext의 인디고(#6366F1) — 사주앱 정체성과 무관, 폐기
❌ 자주색(#8B4B8B) — 무속 부적 톤으로 교체
❌ MZ 핑크/별빛 다크/Apple 미니멀 — 모두 거부됨


## 컴포넌트 스타일 가이드

### 점수 헤더 (Score Header)

- 화면 상단 헤더, 높이 약 80px
- 좌측: 점수 숫자 + 한자 "吉" + 점수바
- 우측: "오늘의 흐름" 라벨 + 사람 전환 버튼
- 배경: 베이지 한지, 보더 없음

### 운세 본문 카드 (Fortune Body Card)

- 화면의 약 40% 차지
- 배경: `--color-paper-light` (밝은 한지)
- 보더: `1px solid --color-paper-dark`
- 패딩: `--space-2xl` (32px)
- 제목: 한자 + 한국어 (예: "吉中之吉 — 관조의 흐름이 도는 날")
- 본문: 4-5줄, 행간 1.7

### 카테고리 카드 (Category Card) — 부적 미니 도장

- 2x2 그리드, 4개 카드
- 배경: `--color-paper-light`
- 상단에 작은 부적 인장 (한자 라벨: 財/愛/職/健)
- 큰 한국어 라벨 (재물운/연애운/직장운/건강운)
- 큰 점수 숫자 + 한 줄 조언
- 보더: `--color-bujeok` 1px, 약간 각진 느낌

### 행운 뱃지 (Lucky Badge)

- 하단 4개 미니 부적 (색상/숫자/방향/소품)
- 작은 사각형 부적 형태
- 배경: `--color-paper-light`, 보더: `--color-bujeok`

### 탭바

- 배경: `--color-paper-dark`
- 활성 탭: 부적 적색 인디케이터 + 굵은 글씨
- 비활성: 먹색 흐림

### 일주 도입부 (Ilju Intro)

- 작은 한자 인장 (예: "甲子") + 한국어 해석 한 줄
- 본문 카드 위에 미니 헤더처럼 표시

## 다크모드 (v1.1 백로그)

- 기본은 라이트 모드만 (한지/먹 분위기 유지)
- 다크모드 = 어두운 신당 분위기 (검정 + 황금 + 빨간 등불)로 별도 시스템 가능
- 출시 후 사용자 요청 시 추가

## 안티 패턴 (절대 사용 금지)

- ❌ MZ 이모지 (💰💕💼🏃) — 부적 한자 인장으로 대체
- ❌ Material Design 아이콘 (outlined circle) — 한자 또는 부적 모티브
- ❌ 그라디언트 배경 (보라/핑크) — 한지 단색
- ❌ 둥근 큰 모서리 (24px+) — 사각 또는 12px
- ❌ "Wealth Luck" 영어 라벨 — 무조건 한국어 ("재물운")
- ❌ 캐릭터 마스코트 (곰돌이/꽃) — 부적/한자만
- ❌ 화려한 그림자 (보라 글로우) — 종이 떠있는 미세 그림자만
- ❌ "Apple Minimal" 같은 영어 텍스트 화면 노출

## 카피 톤

- 본문: ~해요 / ~네요 (해요체)
- Alert: 본문과 동일하게 해요체 통일
- 명리학 용어 (壬子, 정관, 십신) 사용자 화면 노출 금지
- 한자는 부적 모티브로만 사용 (운세 본문에는 한국어만)

## 구현 우선순위

1. **theme.ts 단일화** (ThemeContext 인디고 제거, 부적 컬러 적용)
2. **DailyFortuneScreen 부적 디자인 적용** (가장 자주 보이는 화면)
3. **OnboardingScreen 적용** (첫인상 결정)
4. **30+ 나머지 화면 일괄 적용** (토큰 교체로 자동)

## 검증 기준

- 화면 5초 보고 "사주/운세 앱"으로 즉시 인지되는가
- 한국 30~50대 사용자가 신뢰감 느끼는가
- "오늘의 운세를 부적으로 받아보는" 느낌이 있는가
- 카드 그리드 SNS 느낌 0
